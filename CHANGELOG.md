# Changelog

All notable changes to the Public Health Data Catalog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Initial release of Public Health Data Catalog
- 91+ authoritative public health data sources
- 12 major categories of data sources
- Real-time search functionality across all content
- Category filtering system (6 filters: Global Orgs, US Federal, Disease-Specific, Surveillance, Genomic, APIs)
- Futuristic UI with custom animations
- Parallax scrolling effects
- Glass morphism design elements
- Floating particle effects
- Responsive mobile design
- Sticky search header
- Live results counter
- "No results" message for empty searches
- Back-to-top button
- Scroll progress indicator
- Clickable URLs throughout

### Categories Included
- Global & International Organizations
  - WHO (3 resources)
  - ECDC (2 resources)
  - PAHO (2 resources)
  - UN & World Bank (3 resources)
- US Federal Sources
  - CDC (6 resources)
  - NIH (4 resources)
- Disease-Specific Platforms
  - COVID-19 (3 resources)
  - Influenza (2 resources)
  - Malaria (1 resource)
  - Cancer (2 resources)
- Real-Time Surveillance & Participatory Platforms
  - Outbreak Detection (3 resources)
  - Participatory Surveillance (3 resources including Global Flu View)
- Data Aggregators & Research Platforms
  - Major Data Catalogs (4 resources)
  - Research Data Repositories (3 resources)
- Genomic & Molecular Data (3 resources)
- APIs & Programmatic Access (5 resources)

### Featured Platforms
- **Global Flu View** - International participatory surveillance (11 countries, 22M+ data points)
- **Johns Hopkins CSSE COVID-19** - Most widely used COVID-19 dataset
- **WHO Global Health Observatory** - 1,000+ health indicators
- **CDC WONDER** - Comprehensive US health statistics
- **GISAID** - 15M+ SARS-CoV-2 genomes
- **Global Health Data Exchange** - 10,000+ data sources

### Technical Features
- Single HTML file (self-contained)
- Vanilla JavaScript (no dependencies)
- CSS3 animations and transitions
- Custom Google Fonts integration
- Mobile-responsive breakpoints
- Cross-browser compatible
- Offline capable
- ~120KB file size

### Documentation
- Comprehensive README.md
- Contributing guidelines (CONTRIBUTING.md)
- MIT License
- .gitignore for clean repository
- BibTeX citation format
